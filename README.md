# starter

Ghost博客现代主题。

特别感谢 [byfortress](http://byfortress.com/downloads/starter)，这个主题是基于它做的修改，他的主题可以免费下载。

## 功能
- gitalk评论
- 增加归档页面
- 标签筛选文章
